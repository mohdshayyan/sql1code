1) Variable: It is a storage area where values are stored.All variables are having some names.
Rules of variable naming will be as per the identifier rules.
 e.g.
 a=10     Here a is variable
 c=a+b    here a,b,c are variables

2) Constants/literals:  are the fixed values which cant be changed.

   a)numerical literal:  
     integer literal: numerical values without
                decimal point.  e.g. 0,90,-56
     real/float literal: numerical values with
                decimal point.  e.g. 1.8,-4.5

   b) Boolean literal: True,False values are  
                called boolean literals.
            Boolean literal are the result of  a
           condition.

   e.g.   if age>=90:   Here age>=90 can give 
                          True or False
   c) string/character/text literal: collection of
characters(can be mixture also) which are enclosed in single or double quotes.
   e.g.   
  "aman" ,  'aman',  'a-123, vivek vihar'

 x='ram' Here x is variable and 'ram' is string
 constant.
Input/Output in Python:
   It is must used portion in all the programs of python.
 
 Output in Python: print() function will be used.
 Syntax: 
 1) To print the fixed text:
       print('text to be printed')
 2) To print the value of variable.
       print(variablename)
 3) To print the value of multiple variables.
       print(var1,var2,var3,....)

 Note:a) print() function will move the cursor to next
line after its completion of execution.
     b) when multiple values are printed then
space will be given after every value (till last value gets printed)
